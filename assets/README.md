# Assets

Run all cells in `template.ipynb` from the `projects/proj04/` directory. The notebook will write these plotly HTML files to this folder:

- `protein-distribution.html` — univariate protein histogram
- `protein-missingness-plot.html` — avg_rating by protein missingness
- `fairness-permutation.html` — permutation test for fairness

Copy the `website/` folder contents (README.md and assets/) into your GitHub Pages repo root for Nutrition_and_Ratings.
